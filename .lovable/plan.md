

# Fix Password Reset Email — Branding + Redirect

## Problems Found

### 1. Sender name shows "quotrai" instead of "Foreman"
Line 39 of `auth-email-hook/index.ts`:
```
const SITE_NAME = "quotrai"
```
This is used in the `from` field (line 259): `${SITE_NAME} <noreply@${FROM_DOMAIN}>` — so emails arrive from "quotrai" instead of "Foreman".

### 2. Reset link redirects to lovable.dev
The `confirmationUrl` passed to the template comes from `payload.data.url` (line 225). This URL is generated by the auth system based on the project's site URL configuration. The `ForgotPassword.tsx` correctly sets `redirectTo: window.location.origin + '/reset-password'`, but the auth system may be rewriting the base URL to the lovable.app/dev domain. The `SAMPLE_PROJECT_URL` on line 49 is also set to `"https://quotrai.lovable.app"` — confirming the old project name is baked in.

### 3. Email background appears dark grey in Gmail
The template itself has `backgroundColor: '#ffffff'` (white) — this is correct. The dark appearance in the screenshot is Gmail's dark mode inverting the email. This is not a bug in the template.

## Fix

### Update `supabase/functions/auth-email-hook/index.ts`

1. Change `SITE_NAME` from `"quotrai"` to `"Foreman"` (line 39)
2. Change `SAMPLE_PROJECT_URL` from `"https://quotrai.lovable.app"` to the correct Foreman preview URL (line 49)
3. Change `ROOT_DOMAIN` to `"foreman.world"` if that's the production domain (line 41 — currently `"foreman.ie"`)

These three changes fix the sender name and preview data. The template content itself is already correct (Foreman branding, green button, white background).

### Redirect issue
The redirect URL problem is controlled by the auth system's site URL setting. The `redirectTo` in `ForgotPassword.tsx` is correct (`window.location.origin/reset-password`). Need to verify whether the Lovable Cloud auth site URL configuration is pointing to the right domain. If not, this may need a `configure_auth` call to set the correct site URL.

After the changes, redeploy `auth-email-hook` edge function.
