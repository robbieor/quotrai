

## Update Stripe Secret Key

**What we'll do:** Replace the current Stripe secret key with your new restricted key (`rk_live_...`).

**Steps:**
1. Use the secret update tool to prompt you to paste your new restricted key
2. The key will be securely stored and available to all backend functions (Connect onboarding, invoices, etc.)

**After updating**, the Connect setup flow in Settings → Online Payments should work end-to-end.
