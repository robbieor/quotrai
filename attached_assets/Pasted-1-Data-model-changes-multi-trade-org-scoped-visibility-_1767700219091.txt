A) Server-side API filter (minimum)

All template endpoints accept orgId and the backend resolves trade type:

GET /templates → returns templates for org’s trade only

GET /templates/:id → only if template.trade_type matches org.trade_type

If you’re not using Supabase, implement the same check in your API layer.
